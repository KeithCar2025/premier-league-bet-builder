# Premier League Bet Builder

This project is designed to help users build bets for Premier League matches in an easy and efficient manner. 

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) - Your development environment should have Node.js installed.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/KeithCar2025/premier-league-bet-builder.git
   ```
2. Navigate into the project directory:
   ```bash
   cd premier-league-bet-builder
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Usage
To run the project:
```bash
npm start
```
